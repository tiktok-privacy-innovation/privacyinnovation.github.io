### Running the PPAM protocol

Here we give a simple example to run our protocol. Please refer to [Example running PPAM](example/README.md) for more details.

To run Party A

```shell
cd ${PPAM}/build/example/scripts
bash sender_test.sh
```

To run Party B

```shell
cd ${PPAM}/build/example/scripts
bash receiver_test.sh
```
